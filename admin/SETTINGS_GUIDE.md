# Settings Panel Guide

## Overview
The enhanced Settings panel provides comprehensive configuration options for your Animora Veterinary Clinic website. All settings are stored locally in your browser and can be easily updated.

Configure your clinic's basic information:
- **Clinic Name**: Your official clinic name displayed across the website
- **Tagline/Slogan**: A catchy phrase that represents your brand
- **Copyright Name**: Name displayed in the footer copyright notice

### üîç SEO Settings
Optimize your website for search engines:
- **Meta Description**: Brief description shown in search results (max 160 characters)
- **Meta Keywords**: Relevant keywords for search engine indexing
- **Google Analytics ID**: Track website visitors and analytics

### üìÖ Appointment Settings
Configure appointment booking preferences:
- **Notification Email**: Where appointment notifications are sent
- **Max Daily Appointments**: Limit the number of daily bookings
- **Default Duration**: Standard appointment length in minutes
- **Require Deposit**: Toggle whether deposits are needed for bookings

### üîî Notification Settings
Control which activities trigger email notifications:
- New appointment bookings
- Contact form submissions
- Patient reviews

### üì± Social Media Links
Update social media profile URLs displayed in the footer:
- Facebook
- Twitter/X
- Instagram
- LinkedIn

### üîß Maintenance Mode
Temporarily disable public website access:
- **Enable/Disable**: Toggle maintenance mode
- **Custom Message**: Message shown to visitors during maintenance
- **Note**: Admin panel remains accessible during maintenance mode

### üíæ Backup & Data Management
Export your website data for backup:
- **Staff Data**: Export all team member information
- **Services Data**: Export all service listings
- **Contact Data**: Export contact information
- **Full Backup**: Export everything including all settings

**Recommended**: Export full backup weekly or before major changes

### üîí Security Settings
Change admin password with enhanced security:
- **Requirements**: 12+ characters, uppercase, lowercase, numbers, special characters
- **Strength Indicator**: Real-time password strength feedback
- **Validation**: Prevents weak passwords and ensures confirmation match

## Best Practices

1. **Regular Backups**: Export full backup weekly
2. **SEO Updates**: Update meta descriptions when services change
3. **Password Security**: Change password every 3-6 months
4. **Test Maintenance**: Test maintenance mode in off-hours
5. **Monitor Analytics**: Check Google Analytics regularly to track performance

## Tips

- All settings are saved locally in your browser's localStorage
- Settings persist across sessions
- Export data before clearing browser cache
- Use maintenance mode for major updates
- Keep social media links up to date
