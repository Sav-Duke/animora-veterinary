# Animora AI Backend

Backend API for Animora Farm Health AI Assistant.

## Environment Variables

Required environment variables in Render:

```
MONGO_URI=<your_mongodb_connection_string>
OPENAI_API_KEY=<your_openai_api_key>
PORT=4001
```

## Deployment

This backend is deployed separately from the main website.

Deploy URL: https://animora-backend.onrender.com
